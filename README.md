## Bem-vindo!

![](logo/a.png) HealthDB (_Health Database_) é um sistema de gerenciamento de banco de dados cuja linguagem de definição de dados é a [ADL](http://www.openehr.org/releases/AM/latest/docs/ADL2/ADL2.html) e cuja linguagem de manipulação de dados é a [AQL](http://www.openehr.org/releases/QUERY/latest/docs/AQL/AQL.html). O HealthDB tem como objetivo facilitar o desenvolvimento de Sistemas de Informação em Saúde (SISs) interoperáveis.
